config/language.json

V configuračním souboru je nutné nastavit výchozí jazyk. 

{
    "fallback": "cs"
}

Přes DI si v Listeneru nebo Controlleru natáhneme závislost na "\Gephart\Language\Language": 

$container = new \Gephart\DependencyInjection\Container();

// ... \Gephart\Language\Language $language jako parametr v konstruktoru

echo $language->get(); // cs (fallback)

$language->set("en");
echo $language->get(); // en
