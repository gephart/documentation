Instalace

Pro instalaci se doporučuje použít composer, který vytvoří i základní strukturu aplikace.

Pokud chcete nainstalovat framework do složky "blog", stačí spustit příkaz:

$ composer create-project gephart/gephart blog "0.4"

Práva pro zápis

Po vytvoření základní struktury je potřeba nastavit práva pro zápis ve složce "blog/cache" a "blog/src" kam se ukládá vygenerovaný kód: 

$ cd blog
$ chmod -R 777 cache
$ chmod -R 777 src

Připojení k databázi

Pokud budete používat přibalené ORM, je ještě potřeba nastavit připojení k databázi v souboru "config/orm.json": 

{
  "driver": "mysqli",
  "hostname": "localhost",
  "port": 3306,
  "username": "root",
  "password": "",
  "database": "gephart"
}

Nyní je vše hotovo a stačí si zobrazit přes server složku "blog/web", která jako jediná by měla být přístupná z webu.
